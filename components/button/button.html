Buttons

The Button component is used to trigger an action or event, such as submitting a form, opening a dialog, canceling an action, or performing a delete operation.

Usage of buttons

Buttons can be used in 6 available styles, each representing the essence of the action it performs. Simply include class btn alongwith the style/type of button you wish to use (e.g., class="btn btn-primary"

Buttons with icons and floating buttons

Buttons can also have icons inside, making it more informational and also giving it a visual appeal. Floating button is a button that appears on top of the view, that can be used to perform any supplementary action.
