Toast

A toast provides simple feedback about an operation in a small popup. It only fills the amount of space required for the message and the current activity remains visible and interactive. Toasts automatically disappear after a timeout. In the demo below, buttons are used to trigger the toasts, which are separate from the toast component. Buttons can be created as per requirement. Use class toast to use this component.
