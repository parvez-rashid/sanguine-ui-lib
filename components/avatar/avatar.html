Avatar

An avatar can be used to display the user's profile picture in various contexts, examples of which include the profile page in social media applications, blogs, and navbar.

Image avatars

Avatar can be of 4 different sizes, i.e., large, medium, small, and extra-small. Simply include class avatar along with class avatar-lg, avatar-md, avatar-sm, or avatar-xs according to your size requirement.

Text avatars

The user's initials can also be used in an avatar. Include class avatar-text along with aforementioned size classes as per your requirements (e.g., class="avatar avatar-text avatar-lg").
