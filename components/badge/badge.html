Badges

Badges are used to indicate notifications or user activity status.

Usage of status badges

We can have badges on avatars showing the user's activity status. You can use class badge-online, badge-away, badge-busy, or badge-offline to indicate user status, alongwith generic class badge. Badge can also be had in various sizes according to avatar size; use classes badge-lg, badge-md, badge-sm, or badge-xs.

Number badges

Number badges can be used to indicate notification count. Simply include class number-badge to use this.
